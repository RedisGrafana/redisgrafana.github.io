    Redis plug-ins for Grafana¶

    The Redis plug-ins for Grafana allow users to connect to the Redis databases and build dashboards in Grafana to easily monitor Redis and application data. Data Source can connect to any Redis database On-Premises or in the Cloud.

    Plug-ins are registered in the Grafana repository and available to Grafana, Grafana Enterprise and Grafana Cloud.

    Where Next?¶

    • The Configuration page explains how to configure Redis Data Source
